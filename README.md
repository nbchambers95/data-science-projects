- 🔗 **Project page:** [milkshake-forecasting/README.md](milkshake-forecasting/README.md)
- 📓 **Notebook:** [milkshake-forecasting/MilkshakeModeling.ipynb](milkshake-forecasting/MilkshakeModeling.ipynb)
